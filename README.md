# 🧠 Habit Tracker PRO

A modern and minimal **Habit Tracker** built with **HTML, CSS, and Vanilla JavaScript**.
Track your daily habits, stay consistent, and visualize your progress with a clean dark UI.

## ✨ Features

* ✅ Add new habits with categories
* 🔁 Toggle habits (complete / active)
* ❌ Delete habits
* 🧹 Clear all completed habits
* 🎯 Filter habits (All / Active / Completed)
* 📊 Progress tracking with percentage
* 🔵 Animated progress ring (conic-gradient)
* 💾 Data saved in **localStorage**
* 🌙 Modern dark UI design

---

## 🧱 Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Custom Properties)
* Vanilla JavaScript (ES6+)
* LocalStorage API

---

## 🖥️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/your-username/habit-tracker-pro.git
```

2. Open the project folder:

```
cd habit-tracker-pro
```

3. Open `index.html` in your browser.

## ⚠️ Notes

* This project uses **localStorage**, so your habits are saved in your browser.
* If you clear browser data, habits will be removed.
* No backend is used (fully frontend project).

---

## 💡 Customization

You can easily customize:

* Colors (via `:root` in CSS)
* Categories in the select input
* UI spacing and layout
